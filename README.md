# To do pig

[English version](./doc/en.md)

![Preview](./doc/img/preview.gif)
## 介绍
这是一款 To Do List 软件， 但是加入可爱的小猪提醒功能。灵感来源 《点燃我，温暖你》中男主角李峋使用到的任务提醒小猪。
## 快速开始
```shell
# 安装依赖
yarn
# 开发
yarn dev
# 构建
yarn build
```
## 技术栈
- TailWind Css
- Naive UI
- Vue
- Typescript
- Electron

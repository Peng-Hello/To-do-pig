## Introduction
This is a To Do List software, but add cute pig reminder function. The inspiration comes from the task used by the hero Li Jian in "Light Me, Warm You" to remind the pig.
## Quick start
```shell
# Installing dependencies
yarn
# Develop
yarn dev
# Build
yarn build
```
## Technology stack
- TailWind Css
- Naive UI
- Vue
- Typescript
- Electron
